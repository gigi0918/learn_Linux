## Shell輸入輸出

```>```將執行內容寫進清空的file
```
$ command > file
```  
```>>```將執行內容寫進file的末尾
```
$ command >> file
```  

```2>```將執行錯誤的顯示文字輸出到err
```
$ errcommand >> file 2> err
```  

```<``` 重定向
