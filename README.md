### 中間命令

 命令 | 功能 | 參數 | 功能||
|:---:|:---:|:---:|:---:|:---:|
|grep|顯示檔案和輸入流中和參數配對的行|||可辨別正規表達式 ex:'.'or'.*'|

### grep

顯示檔案 /etc/passwd 中，包含 gigi 的所有行 :
```
$ grep gigi /etc/passwd
```
